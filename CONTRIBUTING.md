# Guia de Contribuição - SinAgro

Obrigado por considerar contribuir com o SinAgro! 🎉

## Como posso contribuir?

### 🐛 Reportar Bugs
- Use o template de **Issue** para bugs
- Descreva o problema com passos para reproduzir
- Inclua logs e screenshots se possível

### 💡 Sugerir Funcionalidades
- Abra uma **Issue** com o label `enhancement`
- Descreva a funcionalidade e seu benefício

### 📝 Melhorar Documentação
- Corrija erros ortográficos
- Adicione exemplos
- Traduza trechos

### 💻 Código
1. Fork o projeto
2. Crie uma branch descritiva
3. Siga o padrão de código existente
4. Adicione testes quando possível
5. Abra um Pull Request

## Padrões de Código

### PHP
- Use PSR-12 como guia de estilo
- Comente funções complexas
- Use tipos declarados quando possível

### CSS
- Use classes semânticas
- Mantenha o código organizado
- Prefira variáveis CSS

### JavaScript
- Use `const` e `let` (evite `var`)
- Comente funções complexas
- Mantenha funções pequenas

## Processo de Pull Request

1. Atualize o README se necessário
2. Atualize o CHANGELOG.md
3. Certifique-se que os testes passam
4. Peça revisão de pelo menos um membro da equipe

Obrigado por contribuir! 🌾
