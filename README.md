## 📋 Sobre o Projeto

**SinAgro** é um sistema de gestão rural desenvolvido para atender pequenos produtores brasileiros que ainda gerenciam suas propriedades de forma manual. A plataforma centraliza informações de **produção, finanças, estoque, equipamentos e animais** em um único ambiente digital, permitindo que o produtor tome decisões baseadas em dados concretos.

### 🎯 Propósito

> *"O esforço existe, mas a informação confiável não acompanha a operação."*

O sistema nasce para transformar a realidade de produtores que dependem de anotações em cadernos, paredes de galpões ou da memória para gerenciar seu negócio. Com o SinAgro, o processo decisório deixa de ser improvisado e passa a ser orientado por evidências.

## ✨ Funcionalidades Principais

| Módulo | Funcionalidades |
|--------|-----------------|
| 🔐 **Autenticação** | Cadastro, login, recuperação de senha e sessão segura |
| 📊 **Dashboard** | Indicadores, gráficos e resumo da propriedade |
| 🏠 **Propriedade** | Cadastro de fazendas, áreas e setores |
| 🌱 **Plantação** | Culturas, ciclos, insumos, colheita e produtividade |
| 🐄 **Animais** | Espécies, lotes, eventos sanitários e reprodutivos |
| 💰 **Financeiro** | Receitas, despesas, extratos e fluxo de caixa |
| 📦 **Estoque** | Controle de insumos, alertas de estoque crítico |
| 🔧 **Equipamentos** | Máquinas, manutenções e status operacional |

## 🚀 Tecnologias Utilizadas

### Backend
- **PHP 8.0+** - Linguagem principal
- **MySQL 8.0+** - Banco de dados relacional
- **PDO** - Prepared statements para segurança

### Frontend
- **HTML5 / CSS3** - Estrutura e estilização
- **JavaScript** - Interatividade e validações
- **Chart.js** - Gráficos e indicadores

### Ferramentas
- **Git & GitHub** - Versionamento
- **Docker** - Ambiente de desenvolvimento (opcional)
- **phpMyAdmin** - Gerenciamento do banco
